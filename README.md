# 星禧水晶 (Xingxi Crystal)

周易五行水晶手串跨境电商独立站

## 项目简介

星禧水晶是一个基于Medusa框架构建的跨境电商平台，专注于销售周易五行水晶手串。

### 主要特性

- 🛍️ 完整的电商功能（产品展示、购物车、订单管理）
- 💳 多种支付方式（PayPal、支付宝、微信支付）
- 🌍 中英文双语支持
- 📱 响应式设计，支持移动端
- 🔮 五行水晶分类系统（金木水火土）
- 📦 基于Medusa v2的强大后端

## 技术栈

### 后端
- Medusa v2.x
- PostgreSQL 14+
- Redis 7+
- Node.js 18+

### 前端
- Next.js 14+ (App Router)
- Tailwind CSS
- TypeScript
- Medusa UI / shadcn/ui

### 支付集成
- PayPal
- 支付宝 (Alipay)
- 微信支付 (WeChat Pay)

## 产品分类

### 五行水晶体系

1. **金系水晶** - 白水晶、白幽灵、银发晶
2. **木系水晶** - 绿幽灵、绿发晶、东陵玉
3. **水系水晶** - 黑曜石、黑发晶、茶晶
4. **火系水晶** - 红玛瑙、石榴石、红发晶
5. **土系水晶** - 黄水晶、黄虎眼、钛晶
